# Stock-MA-Tracker

A configurable stock moving-average strategy tracker and Telegram notifier.

## Initial strategy

The first strategy version uses:

- QQQ as the signal instrument
- TQQQ as the target instrument
- QQQ SMA200 as the trend indicator
- SMA200 × 1.04 as the Risk-On threshold
- SMA200 × 0.97 as the Risk-Off threshold
- Previous-state retention inside the neutral zone
